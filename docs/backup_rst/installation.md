## IIIIIIIIII

If you want to contribute to the project or need the latest development
version:

1.  Clone the repository:

    ``` bash
    git clone https://github.com/kasnycdev/MyJobSpyAI.git
    cd MyJobSpyAI
    ```

2.  Install in development mode:

    ``` bash
    pip install -e .
    ```
